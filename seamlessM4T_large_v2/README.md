## SeamlessM4T large v2 evaluation results:

- `asr_fleurs_test.csv`: ASR WER/CER evaluated on FLEURS test set
- `s2st_cvss_c_test.csv`: ASR-BLEU scores evaluated on CVSS-S test set for the task of S2ST. Transcriptions for ASR-BLEU are generated with Whisper-large.
- `s2st_fleurs_test.csv`: ASR-BLEU scores evaluated on FLEURS test set for the task of S2ST. Transcriptions for ASR-BLEU are generated with Whisper-large.
- `s2tt_covost2_test.csv`: BLEU scores evaluated on CoVoST2 test set for the task of S2TT.
- `s2tt_fleurs_test.csv`: BLEU scores evaluated on FLURS test set for the task of S2TT.
- `t2st_fleurs_test.csv`: ASR-BLEU scores evaluated on FLEURS test set for the task of T2ST. Transcriptions for ASR-BLEU are generated with Whisper-large.
- `t2tt_flores_devtest.csv`: ChrF2++ scores evaluated on FLORES devtest set for the task of T2TT (MT).

For the exact test set ids, see [https://dl.fbaipublicfiles.com/seamless/metrics/evaluation_data_ids.zip](https://dl.fbaipublicfiles.com/seamless/metrics/evaluation_data_ids.zip)

For the exact metrics signatures, see the *Metric card* (Appendix J of Seamless Communication et al, Seamless: Multilingual Expressive and Streaming Speech Translation, Arxiv, 2023)
